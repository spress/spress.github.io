Spress 2.0.0 released

After ten months working on the new major version of Spress at last is out. The code of Spress 2.0.0 has been written from the ground up to incorporate new features like data sources or generators . This version marks the end-of-life for Spress 1.x. See the migration page for more details.

The list of the most important changes since RC are available at the changelog page. The documentation for Spress 2 is available at docs pages. Documentation for Spress 1.x is available here.

Key features

  • A new site structure.
  • Collections.
  • Data sources.
  • Generators.
  • Renderizers.
  • The data writer.
  • Command plugins.
  • A new way for paginating items.
  • Taxonomies.
  • Parsedown converter for Markdown content.
  • self-update command.

Plugins and themes

The following themes has been ported to Spress 2:

  • Spresso. This theme are included in spress.phar out of the box.
  • Clean blog.
  • About me.
  • Directive.

The following plugins has been ported to Spress 2:

  • Metadata of a Github repository.
  • More tag.
  • Data loader.

All plugin & theme developers are encouraged to upgrade to this version.

How to get this version?

To update from Spress 2.0.0-rc:

spress self-update

To get this release directly:

$ curl -LOS https://github.com/spress/Spress/releases/download/v2.0.0/spress.phar

Enjoy it!

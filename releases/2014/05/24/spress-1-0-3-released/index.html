Spress 1.0.3 released

Spress 1.0.3 is available with one stability fix (thank you to @enzolutions) and ConfigServiceProvider component updated to the last version. Now Spress has support to .dist files.

We are involved with the next Spress 1.1.0. Today the roadmap was published.

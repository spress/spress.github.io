Extending Twig Developers

Creating a plugin Lifecycle Events list Converters Extending Twig IO API

The spress.start event enables you to extend Twig with new functions, tags and tests.

Add a filter

$subscriber->addEventListener('spress.start', 
    function(EnviromentEvent $event)
    {
        $event->addTwigFilter('dummy', function($string){
            return $string . ' (is dummy) ';
        });
    });

Usage in template: 

{{ 'The text' | dummy }}

Add a function

$subscriber->addEventListener('spress.start', 
    function(EnviromentEvent $event)
    {
        $event->addTwigFunction('superf', function($string){
            return strtoupper($string);
        });
    });

Usage in template: 

{{ superf('the text') }}

Add a test

$subscriber->addEventListener('spress.start', 
    function(EnviromentEvent $event)
    {
        $event->addTwigTest('red', function($val){
            return $val == 'red';
        });
    });

Usage in template: 

{% if my_value is red %}
...
{% endif %}
